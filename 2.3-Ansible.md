# BR-SRV

Устанавливаем `ansible`

`apt install ansible`

Создаём ключи RSA

ssh-keygen -t rsa

Копируем открытый ключ RSA на HQ-SRV

`ssh-copy-id -p 2024 sshuser@192.168.100.2`

![{CCC7F208-EC49-4E8C-94B0-37052BE844A1}](https://github.com/user-attachments/assets/e37219a8-dce9-4f27-a1a9-b7834467a6d2)

На HQ-CLI

`ssh-copy-id debian@192.168.100.2` 

![{1E61C5FD-4084-4072-8517-4F672D2D6840}](https://github.com/user-attachments/assets/ffaa4208-4918-4a9b-8827-595aa0479d81)

На HQ-RTR

`ssh-copy-id net_admin@172.16.4.2`

![{20AEAD49-AA73-471A-95D6-E167EBB43129}](https://github.com/user-attachments/assets/4be5ab0e-21c3-400a-add6-c00a395925db)

На BR-RTR

`ssh-copy-id net_admin@172.16.5.2`

![{EB4817A5-FE29-46F4-BF5A-2F9B9A6A9EDD}](https://github.com/user-attachments/assets/98cb1caa-6c9b-45c1-89a9-a5adf2c7706a)

Создаём директорию для ansible

`mkdir /etc/ansible`

Создаём файл инвентаря и прописываем конфигурацию

`nano /etc/ansible/demo`

![{EC25C867-A6AB-4837-A268-5D88ACD33C01}](https://github.com/user-attachments/assets/52297625-08f5-42f6-abcf-4b73dcc987d7)

Проверяем, всё должно быть зелёное

`ansible all -i /etc/ansible/demo -m ping`
