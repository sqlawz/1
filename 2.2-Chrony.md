# HQ-RTR

Устанавливаем `chrony`

`apt install chrony`

Добавляем конфигурацию и комментируем подчёркнутую строку

`nano /etc/chrony/chrony.conf`

Перезапускаем службу

`systemctl restart chrony`

# Проверка 

После добавления всех конфигураций, перезапускаем службу `chrony` на `HQ-RTR`

`systemctl restart chrony`

Проверяем соединение с клиентами на HQ-RTR

`chronyc clients`

![{7C98E1B5-F55E-450A-BCE5-A45BF9C799B7}](https://github.com/user-attachments/assets/580b9f15-7054-4b9e-9288-6ac6137ab878)

Проверяем соединение от клиента к серверу, например на `BR-SRV` ( слева должна быть `*`, если `?`, то соединения нет )

`chronyc sources`
