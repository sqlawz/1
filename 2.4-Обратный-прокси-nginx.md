Устанавливаем nginx на `HQ-RTR`

`apt install nginx`

Создаём файл для обратного прокси

`nano /etc/nginx/sites-available/rp`

```
server {
        listen 80;
        server_name moodle.au-team.irpo;

        location / {
        proxy_pass http://192.168.100.2:80;
        proxy_set_header Host $host;
    }
}

server {
        listen 80;
        server_name wiki.au-team.irpo;

        location / {
        proxy_pass http://192.168.0.2:8080;
        proxy_set_header Host $host;
        }
}
```

Создание символической ссылки

`ln -s /etc/nginx/sites-available/rp /etc/nginx/sites-enabled/`

Перезапуск `nginx`

`systemctl restart nginx`

# После установки `Wiki` и `Moodle`, переходите по соответствующим адресам в браузере

`http://wiki.au-team.irpo`

`http://moodle.au-team.irpo`
